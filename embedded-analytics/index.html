Choose the embedding option best-suited to your analytics needs

React SDK: The most customizable option

Build analytics applications with a custom frontend, or natively embed GoodData dashboards and insights directly into your React app.

  • Provides the best developer experience.
  • Best for React-built applications and a fully end-user aligned UX.

Read more about React SDK

Web Components: The most efficient option

Embed dashboards and visualizations natively into any non-React app. Web components are specifically designed to work with all modern web browsers.

  • Provides the best compromise between ease of setup and flexibility.
  • Best for Angular, Vue, or any other non-React app.

Read more about Web Components

iFrame: The quick and easy option

Embed dashboards easily by copy-pasting a simple HTML code snippet, without deep integration and customization requirements.

  • Provides the easiest setup.
  • Best for simpler integrations or for testing initial proof-of-concepts.

Read more about iFrame
