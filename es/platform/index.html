Presentamos FlexQuery

FlexQuery es el nuevo motor de análisis que impulsa GoodData. Ahorre dinero, mejore el rendimiento y garantice la gobernanza con nuestra tecnología de vanguardia en memoria.

Ahorro de costos

El motor de caché de FlexQuery, FlexCache, almacena automáticamente conjuntos de resultados completos y parciales para su reutilización, ahorrándole una cantidad significativa de dinero en su arquitectura de datos en la nube.

Descubra cuánto puede ahorrar

Integración de datos

Las capacidades integradas de transformación y federación de FlexQuery facilitan la combinación de datos de todas sus aplicaciones a través de SQL o APIs. Puede materializar datos en memoria o en disco, o consultarlos en vivo desde sus bases de datos, e incluso combinar ambos modos.

Rendimiento

FlexQuery soporta cálculos ultrarrápidos en memoria en grandes volúmenes de datos. En el futuro, proporcionaremos controles de caché más granulares para ayudarte a equilibrar la actualidad de los datos frente al ahorro de costos.

Aprenda más sobre rendimiento

Código abierto

FlexQuery está construido sobre las mejores herramientas de analítica de código abierto, incluyendo Apache Arrow, Iceberg y DuckDB, preconfiguradas para casos de uso básicos de inteligencia empresarial (BI). Le proporcionamos acceso para que pueda desarrollar sus propios módulos de FlexQuery y así abordar necesidades de ingeniería de datos o ciencia de datos.

Aprenda sobre herramientas de código abierto

Analytics Lake

A diferencia de un lago de datos, el Analytics Lake reúne en una sola plataforma datos, metadatos, modelos de datos, métricas e informes/visualizaciones. Esto facilita la creación y gestión de cualquier tipo de aplicación analítica sin enfrentar complejos desafíos de integración.

Funciones que los creadores de analíticas adoran

API primero

La funcionalidad de la plataforma y los metadatos pueden ser ejecutados y gestionados a través de API bien diseñadas y APIs abiertas.

Definiciones declarativas

Todas las definiciones de análisis son declarativas y pueden ser exportadas, importadas, versionadas, compartidas y heredadas.

SDKs

Construya integraciones y gestione la plataforma con SDKs de código abierto — Python, React, y JavaScript.

CI/CD

Integre análisis con sus pipelines de CI/CD para automatizar el desarrollo, tal como lo haría con cualquier otro código.

GoodData para VS Code

Aproveche los flujos de trabajo de ingeniería de software para mejorar la versionización, colaboración y automatización con el plugin de VS Code.

Capa semántica

Simplifique datos complejos en métricas reutilizables y con contexto — reestructure sin romper las analíticas.
